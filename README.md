# Internal Tickets - Sisteam de gestión de incidencias TI

## Descripción

Proyecto desarrollado con Django para crear y gestionar tickets de soporte TI.

### Objetivos

1. Aprender Django en un proyecto real (modelos, vistas, admin, auth migraciones).

2. Producto mínimo. Registrar, asignar y cerrar tickets; listado y búsqueda básica.

3. Servira como prototipo que pueda adaptarse a procesos internos de soporte.

### Características principales

- Autenticación: Usuarios y roles básicos (soporte / solicitante).

- Modelos: Ticket con estado, prioridad, asignado y comentarios.

- Interfaz: Admin de Django para administración rápida.

- Persistencia: Soporte inicial con SQLite; fácil migración a PostgreSQL.
